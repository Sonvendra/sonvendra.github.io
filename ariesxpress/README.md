# AXL Website Clone

This is a static HTML/CSS clone of the AXL Logistics website.

## Files Structure

- `index.html` - Home page
- `about.html` - About AXL page
- `services.html` - Services page
- `contact.html` - Contact page
- `styles.css` - Main stylesheet
- `script.js` - JavaScript for interactive features

## Features

- Responsive design
- Image slider on homepage
- Testimonials slider
- Modal popup for quote request
- Dropdown navigation menus
- Contact form
- Track shipment form

## Usage

1. Create an `images/` folder in the root directory
2. Add all required images to the `images/` folder
3. Open `index.html` in a web browser

## Notes

- All forms are currently set up with basic validation but don't actually submit data (static site)
- Some links point to `.php` and `.aspx` files from the original site - these can be updated to HTML files if needed
- The design closely matches the original AXL website layout and styling
